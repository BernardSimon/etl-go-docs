
# 关于 ETL-GO

ETL-GO 是一个基于 Go 语言的可扩展 ETL 平台，目标是为开发者提供一个轻量级、可配置、可扩展的数据集成工具。

## 设计目标

- 简化数据集成流程
- 降低二开成本
- 支持常见数据库 + 文件 + HTTP 数据源
- 提供可视化 Web 管理界面

## 核心设计思路

### 1. 组件化

平台把 ETL 能力拆分为 `Datasource / Source / Processor / Sink / Executor / Variable` 六类组件，通过工厂注册统一管理。

### 2. 流水线并发执行

执行链路采用 goroutine + channel 方式运行，支持批处理、取消传播和错误快速失败。

### 3. 前后端与 API 双通道

- 提供 Web 管理台满足可视化配置
- 提供 `/api/v1` 接口满足平台集成与自动化调用

## 主要能力

- 数据源管理
- 任务配置
- 数据输入/处理/输出
- 任务调度与手动执行
- 任务记录与日志分析
- 文件资产管理
- 组件工厂扩展机制

## 适用场景

- 数据库同步（库到库）
- 文件导入导出（CSV/JSON）
- HTTP API 拉取或推送
- 定时数据处理任务
- 团队内数据自动化平台搭建
