
# 任务调度与执行

ETL-GO 支持手动执行与 Cron 调度。

## 任务类型

- `manual`：手动触发任务
- Cron 表达式：定时执行任务，例如 `0 0 * * *`。

说明：当前后端使用标准 Cron 解析（5 段格式）。

## 创建定时任务

1. 进入任务创建页面
2. 填写 `Cron` 表达式
3. 保存后任务会自动注册到调度器

## 手动执行

手动任务可以直接点击“立即执行”触发。

## 取消执行

任务正在运行时，可在任务记录页面点击取消。

## 调度状态说明

- `status=1`：任务已加入调度器
- `is_running=true`：任务当前正在执行

## 注意事项

- `manual` 任务无法注册为自动调度任务
- Cron 表达式必须符合标准格式
- 同一任务正在运行时，新周期会跳过并在下次重试

## 常见错误排查

### 1. `manual task cannot be scheduled`

- 任务 `cron` 为 `manual`，不能调用调度接口。

### 2. `invalid cron expression`

- Cron 格式不符合标准，需修正后保存。

### 3. 任务显示调度中但未实际执行

- 检查任务是否一直处于 `is_running=true`。
- 检查上次执行是否阻塞未结束。
