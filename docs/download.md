### 更新日志
```txt
etl-go v0.1.0 发布
我们很高兴地宣布 etl-go 的第一个正式版本发布了！这是一个现代化、高性能、易于使用的开源 ETL（Extract, Transform, Load）工具。
🌟 主要特性
开箱即用：内置多种常用的数据源、处理器和目标组件
高扩展性：插件化架构，支持自定义组件开发
可视化配置：提供 Web 界面进行任务配置和监控
多数据源支持：支持 MySQL、PostgreSQL、SQLite、Doris、CSV、JSON 等多种数据源
丰富的处理器：数据类型转换、行过滤、数据脱敏、列重命名等功能
任务调度：支持定时任务和手动触发
变量管理：动态配置和 SQL 变量支持
🚀 快速开始
下载适合您操作系统的版本并解压（MacOs的Apple Silicon设备请下载macos-arm版本）
运行 etl-go 服务
访问 Web 界面 [http://localhost:8081]（默认用户名：admin，密码：password123）
📦 包含内容
etl-go 可执行文件
默认配置文件 config.yaml
```
